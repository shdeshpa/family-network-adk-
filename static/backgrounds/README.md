# Background Images

Please save your temple background images in this directory with the following names:

1. `temple_street.jpg` or `temple_street.png` - Temple street marketplace scene (Image #1)
2. `temple_blessing.jpg` or `temple_blessing.png` - Temple blessing ceremony scene (Image #2)
3. `temple_garden.jpg` or `temple_garden.png` - Temple garden scenic view (Image #3)

These images will be used as faded background overlays throughout the application.

## Supported Formats
- JPG/JPEG
- PNG
- WebP

The application will automatically detect and use these images with a faded transparent overlay effect.
